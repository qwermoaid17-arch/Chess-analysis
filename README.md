## 📁 Repository Structure
This repository is organized into directories that reflect my progressive architectural understanding of databases:

* 📂 **`01_Basics`**: 
  Establishing secure database connections using the `pymysql` library, creating fundamental tables, and implementing robust Error Handling using `Try/Except` blocks.
  
* 📂 **`02 data_types`**: 
  Engineering applications focused on Memory Optimization. This includes a deep understanding and correct application of numeric data types (`TINYINT`, `SMALLINT`, `INT`, `BIGINT`) to prevent resource waste.

* 📂 **`03_Date_and_Time`**: 
  Professional management of system timestamps. This covers both application-side timing (using Python's `datetime` module) and database-side timing (utilizing built-in functions like `CURRENT_TIMESTAMP` and `NOW()`).

## 🛡️ Core Engineering Principles Applied
1. **SQL Injection Prevention:** Consistent use of parameterized queries (`%s`) to ensure safe data insertion across all scripts.
2. **Resource Management:** Utilizing bulk insertion techniques (`executemany`) to minimize server load and network requests during massive data operations.
3. **Storage Efficiency:** Strict and deliberate selection of data types to prevent unnecessary consumption of disk space and RAM.
